Pokedex APP - A Swift API for Pokémon.

This app was built using Switf 5 with MVC Architecture and uses https://pokeapi.co API.

Features
Displays a list of all the Pokémons available on the API (with lazy loading).
Selecting a pokémon will take you to the pokémon details view.

Setup
Most up-to-date branch: Develop
No special setup required, just download and run the project on Xcode.
No external dependecies were used.
